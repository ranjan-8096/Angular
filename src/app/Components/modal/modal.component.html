Cards

Details for the Cards section are below.

Please do follow the below given instructions while creating your Card component.

  • Title of the card should be less than 15 characters
  • Description of the card should be less than 200 characters
  • Button text of the card should be less than 10 characters
  • It is mandatory for you to specify whether you want card image with (Yes/No)
  • It is mandatory for you to specify whether you want to add buttons with (Yes/No)
  • It is mandatory for you to specify number of buttons to add if you have selected yes to add buttons
  • You can change the default width selected value and specify the width of the card from the given options

Buttons

Details for the Button section are below.

Please do follow the below given instructions while creating your Button component.

  • Button text should be less than 15 characters
  • Border radius should be less than 50px
  • Border width should be less than 5px
  • It is mandatory for you to select the width of the button from the given options
  • It is mandatory for you to specify whether you want border radius with (Yes/No)
  • If you have selected yes for border radius than it is mandatory to speficy the radius
